# Turbofan Engine Degradation Simulation ✈️⚙️

## Overview
This project focuses on predictive maintenance by simulating and predicting the degradation of turbofan jet engines. A technical dashboard was developed to monitor engine health and estimate the Remaining Useful Life (RUL) using advanced machine learning models. 

## Dataset
The models are trained and evaluated on the **NASA C-MAPSS** (Commercial Modular Aero-Propulsion System Simulation) dataset, which contains simulated run-to-failure sensor data from fleet engines under various operational conditions.

## Key Features
* **Predictive Modeling:** Accurately forecasts engine degradation and Remaining Useful Life.
* **Real-Time Diagnosis:** Features a dashboard for continuous monitoring and visualization of engine health metrics.
* **Proactive Maintenance:** Aims to reduce downtime and prevent catastrophic engine failures by alerting users before critical thresholds are reached.

## Tech Stack & Algorithms
* **Deep Learning:** Long Short-Term Memory (LSTM) networks for capturing temporal dependencies in sensor time-series data.
* **Machine Learning:** Random Forest algorithms for robust feature evaluation and predictive modeling.
* **Deployment:** HTML-based interactive technical dashboard.
