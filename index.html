Project Overview

The aviation industry relies heavily on the reliability of jet engines. Traditional reactive maintenance leads to costly downtime. This project introduces a Predictive Maintenance System using Random Forest Classification to forecast engine failures before they occur.

By leveraging historical sensor data (Temperature, Pressure, Vibration) from the NASA C-MAPSS dataset, this application simulates a real-time monitoring environment. It includes an Advanced Cockpit Voice Alert System that mimics Airbus flight deck warnings for immediate situational awareness.
